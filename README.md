# FetchAPIApp

# TALLER – CONCEPTOS FUNDAMENTALES DE PWA – FETCH API

**ACTIVIDADES**

En grupos de 3 integrantes realice las siguientes actividades a través de un proyecto en VSCode que se llame FetchAPIApp que incorpore las siguientes funcionalidades:

1. Muestre el listado de los primeros 30 usuarios que encuentre en forma de tabla que contenga los campos nombre, correo, teléfono, nombre de la compañía y website obtenidas a través de las APIs de la página https://jsonplaceholder.typicode.com/users.

2. Cree un hipervínculo en el nombre que permita en otra parte de la página web ver el detalle del usuario y muestre al menos 10 datos del mismo. Asegúrese de mostrar los datos que se puedan modificar en una caja de texto para que puedan ser modificados en el siguiente punto. Verifique que el cambio se ejecutó a nivel de consola.

3. A partir de la tabla simule la realización de una tarea PATCH que permita cambiar alguno de los datos de un usuario seleccionado. Verifique que el cambio se ejecutó a nivel de consola.

4. A partir de la tabla simule la realización de una tarea DELETE que permita cambiar alguno de los datos de un usuario seleccionado. Verifique que el cambio se ejecutó a nivel de consola.

5. Maneje en su aplicación al menos dos imágenes mediante el método Blob del FetchAPI.

6. Verifique que todas las operaciones mencionadas anteriormente manejan errores.
